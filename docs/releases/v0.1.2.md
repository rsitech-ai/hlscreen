# hlscreen v0.1.2

`hlscreen` is a read-only Rust terminal workstation for recording, replaying,
and screening public Hyperliquid spot market data. It does not connect wallets,
handle private keys, read private account streams, or expose order, cancel, or
withdrawal routes.

## Highlights

- Public tree hygiene completed: internal development tooling
  (`.specify/`, `.agents/`, `specs/`, `third_party/spec-kit/`), internal audit
  reports (`docs/reports/`), and the `.mailmap` personal-email remap are no
  longer part of the public repository.
- Third-party attribution (`THIRD_PARTY_LICENSES.txt`,
  `THIRD_PARTY_NOTICES.md`) regenerated; every packaged Rust dependency
  license and notice is preserved.
- CI runs under a GitHub-owned-actions-only policy by installing `uv` from
  PyPI with a pinned version.
- Native Apple Silicon macOS package with project license, dependency
  licenses, notices, README, changelog, and SHA-256 checksum.

## Install on Apple Silicon macOS

Download `hlscreen-aarch64-apple-darwin.tar.gz` and its adjacent `.sha256`
file, then run:

```bash
shasum -a 256 -c hlscreen-aarch64-apple-darwin.tar.gz.sha256
tar -xzf hlscreen-aarch64-apple-darwin.tar.gz
./hlscreen-aarch64-apple-darwin/bin/hls --help
./hlscreen-aarch64-apple-darwin/bin/hls doctor --data-dir /tmp/hlscreen-v0.1.2
```
